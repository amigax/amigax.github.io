What it does

📁 File activity

Every read, write, create, delete and rename — system-wide, every process, in real time.

🗝 Registry activity

Watch apps query, set and delete registry keys. Find out exactly where software stores its settings.

⚡ Process events

See every process start and exit with parent PID. Know what spawned what.

🌐 Network connects

TCP and UDP connections with destination IP and port. See what's phoning home.

🔍 Live filtering

Filter by process name or path substring. Type and the stream narrows instantly.

⏸ Pause on keyword

Set a path fragment as a breakpoint — auto-pauses the moment it appears.

🎨 Per-process colours

Each process gets its own row colour so you can visually track multiple apps at once.

💾 Save & copy

Export the full log to txt/csv. Double-click any row to copy its path instantly.

Inspired by SnoopDos by Eddy Carroll for the Commodore Amiga (1992) — the original tool that let you see what any program was actually doing. Same idea, same spirit, three decades later.
